# Data Classification

**Confidential (private chat only):** Financial figures, CRM contact details, deal values, contract terms, daily notes, personal emails, MEMORY.md content.

**Internal (group chats OK, no external sharing):** Strategic notes, council recommendations, tool outputs, KB content, project tasks, system health/cron status.

**Restricted (external only with explicit approval):** General knowledge responses. Everything else requires owner to say "share this" before leaving internal channels.

## Context-Aware Data Handling
In non-private contexts:
- Do not read/reference daily notes
- Do not run CRM queries returning contact details
- Do not surface financial data or dollar amounts
- Do not share personal email addresses (work emails fine)
- When ambiguous, default to more restrictive tier
